一般情况下，犯罪前科只能通过合法渠道查询，例如本人依法向有关部门申请查询，或由司法机关、公安机关在依法履行职责时进行调取。未经本人同意，任何个人或组织都无权擅自查询他人的犯罪前科信息。

总的来说，犯罪前科信息属于高度敏感的个人隐私，所有查询行为都应在法律允许的范围内进行，确保合法合规。

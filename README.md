# 公安上网追逃是什么

公安上网追逃，是指公安机关将依法通缉的在逃人员信息录入全国在逃人员信息系统，通过信息化手段协助各地公安机关开展协查和抓捕工作。

被列入上网追逃名单的人员，通常因涉嫌犯罪且未到案，公安机关依法发布协查信息，用于在全国范围内进行查找和控制。相关信息仅限执法机关在法定职责范围内使用。

上网追逃信息属于执法与公共安全范畴，普通个人无法自行查询或操作。如涉及本人情况核实，应通过合法渠道咨询或配合有关部门处理。

总的来说，公安上网追逃是一种依法开展的执法协作机制，目的是维护社会秩序和公共安全。
